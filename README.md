# 🍽️ Nelly Food

Projeto front-end de um site para restaurante, desenvolvido com **HTML**, **CSS** e **JavaScript**.

## 📌 Sobre o projeto

O **Nelly Food** é um projeto de site para restaurante criado para apresentar a marca, destacar pratos
e facilitar o contato com clientes por meio de uma interface visual agradável e organizada.

## ✨ Funcionalidades

- Página inicial
- Navegação entre páginas
- Galeria de imagens
- Botão de pedido pelo WhatsApp com mensagem pronta
- Layout personalizado

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
